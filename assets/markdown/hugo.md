## Coba Hugo

Hugo adalah sebuah SSG yang ditulis dalam bahasa golang.

Kita bisa menulis artikel dengan menggunakan markdown dan hugo akan mengubahnya menjadi file html.

Jadi tetap terasa ngodingnya walaupun cuman nulis postingan.

Dan juga, saya belajar web dari hugo.

Jadi mula-mula saya install hugo, pilih tema yang telah disediakan dan jadi!!

Tapi saya tidak terlalu suka dengan temanya jadi ketimbang mencari tema yang cocok, saya memilih untuk merombak temanya.

Dan dari situ saya jadi belajar HTML, CSS, Javascript, dan beberapa syntax default hugo.
